Proyectos realizados

Sistema de monitoreo de emergencias.

Estado: Finalizado.

Presupuesto: $ 250.000.000

Proyecto realizado bajo solicitud de la Dirección Nacional de Policía, el cual buscaba realizar el monitoreo de emergencias en curso mediante el uso de tecnologías. Entre las principales tecnologías usadas se encuentran: C# .NET, Microsoft Server, Javascript, CSS, Html y React.

Sistema de facturación electrónica.

Estado: Finalizado.

Presupuesto: $ 302.000.000

Proyecto efectuado para una cadena nacional de almacenes, la cual buscaba implementar las nuevas políticas de facturación electrónica determinadas por la Dirección de Impuestos y Aduanas Nacionales (DIAN) bajo la resolción 165 de 2023. Entre las principales tecnologías están: PHP, Laravel, Angular, CSS, Html, Jenkins y NGINX.

Sistema para el registro de notas - Universidad del Caribe.

Estado: Finalizado.

Presupuesto: 175.890.000

Este software realiza el reemplazo del anterior sistema utilizado por la universidad para realizar el registros de notas académicas de todos sus estudiantes a nivel de pregrado y posgrado en todas las sedes de la Universidad del Caribe. Entre las principales tecnologías están: PHP, Laravel, Angular, CSS y Html.
